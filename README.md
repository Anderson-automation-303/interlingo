# 🌐 Interlingo

Interlingo is a Discord translation bot that enables real-time multilingual communication inside Discord servers. It automatically translates messages using external translation services like DeepL to help communities communicate without language barriers.

---

## ✨ Features

- 🌍 Real-time message translation in Discord channels  
- ⚡ Fast and lightweight command handling  
- 🧠 Powered by DeepL translation API  
- 🔒 Minimal data storage (no permanent message storage)  
- 🛠️ Easy server setup via OAuth2 invite  
- 📊 Designed for scalability and future premium features  

---

---

## 🧪 How It Works

1. User sends a message or uses a translation command  
2. Bot processes the message  
3. Content is sent to DeepL for translation  
4. Translated message is returned instantly in Discord
